# 🎮 Rock Paper Scissors

A modern and interactive **Rock Paper Scissors** game built with **HTML, CSS, and JavaScript**. Challenge the computer in a fun, animated battle featuring smooth gameplay, score tracking, sound effects, and responsive design.

---

## ✨ Features

* 🎮 Classic Rock Paper Scissors gameplay
* 🤖 Random computer (bot) opponent
* 🔊 Sound effects

  * Click
  * Win
  * Lose
  * Draw
* 🎯 Live score tracking
* 🎭 Battle animation
* 💥 Winner & loser messages
* ✨ Choice animations
* 📱 Responsive design
* ⚡ Lightweight and fast

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Faizan-khan144/Rock-Paper-Scissors.git
```

### Open the project

Open **index.html** in your browser.

No installation or additional dependencies are required.

---

## 🎮 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The bot randomly selects its move.
3. Watch the battle animation.
4. Scores update automatically after each round.
5. Continue playing as long as you like!

## 📋 Game Rules

* 🪨 Rock beats Scissors
* 📄 Paper beats Rock
* ✂️ Scissors beats Paper
* Same choices result in a draw
